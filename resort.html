Fjordington Resort Webpage Project

For this pair project, Erica and her partner Dylan S. created a website for a fictional ice hotel resort named Fjordington. This site contained multiple webpages with content. Once the basic site template was created, this project was all about experimenting with CSS techniques as well as working with branching in Github.

Several different branches were created in order to experiment with the layout of the content already created within the HTML file pages. The first branch was the "master", and this one was stylized with lighter colors and a light blue background.
