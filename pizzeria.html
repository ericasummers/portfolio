Migranier Pizzeria Webpage Project

For this pair project, Erica and her partner Dylan S. created a website for a fictional pizzeria restaurant named Migranier. This site contained multiple webpages with content. Once the basic site template was created, this project was all about experimenting with CSS techniques as well as working with branching in Github.

Several different branches were created in order to experiment with the layout of the content already created within the HTML file pages. The first branch was entitled "side-menu", and this one was stylized so that the navigation menu bar appeared to the right of the page content.
